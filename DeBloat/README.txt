DeBloat Utility
Heavily cribbed from Tron (https://github.com/bmrf/tron) Stage 2
Modified for usage with Windows 10
Modified for usage with HP machines

Intended for use on fresh Windows 10 installs BEFORE desired applications are installed.
System may reboot automatically during uninstall process.  If this occurs, restart the utility.

Utility checks system GUID list against a list of known unwanted application GUIDs and uninstalls anything it finds.
Additionally, it checks for applications by name.
Removes Windows 10 built-in apps from Microsoft as well as third party OEM apps.

Disables and removes OneDrive.
